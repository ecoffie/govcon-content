## Content Writing Workflow

### For Each Page:

1. **☐ Open** the PRD (`cagecode-programmatic-prd.md`) for structure reference
2. **☐ Check** keywords in `strategy-notes.md` for SEO targets
3. **☐ Write** content in a local `.md` file first
4. **☐ Review** for keyword placement (H1, first paragraph, H2s)
5. **☐ Copy** to Framer CMS fields
6. **☐ Add** images with alt text
7. **☐ Preview** on desktop + mobile
8. **☐ Publish** when ready

### SEO Checklist Per Page:

- ☐ Primary keyword in Title (H1)
- ☐ Primary keyword in first 100 words
- ☐ Primary keyword in at least one H2
- ☐ Primary keyword in meta description
- ☐ Primary keyword in URL slug
- ☐ 5-10 internal links to other GovCon Giants pages
- ☐ 3-5 external links to .gov sites
- ☐ All images have alt text with keywords
- ☐ Meta title is 50-60 characters
- ☐ Meta description is 150-160 characters
